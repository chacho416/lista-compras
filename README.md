# 🛒 Lista de Compras en Vue.js

Una aplicación web sencilla e intuitiva para gestionar listas de compras, desarrollada con Vue.js. Este proyecto permite a los usuarios agregar productos, marcarlos como comprados y eliminarlos de la lista fácilmente.

## 🚀 Funcionalidades

* **Agregar artículos:** Añade nuevos productos a tu lista de compras con un solo clic o presionando 'Enter'.
* **Marcar como comprado:** Tacha los artículos que ya has puesto en el carrito para llevar un mejor control.
* **Eliminar artículos:** Quita productos de la lista si ya no los necesitas o te equivocaste.
* **Almacenamiento local (Opcional/Si aplica):** Guarda tu lista en el navegador (Local Storage) para no perderla si recargas la página.
* **Diseño Responsivo:** Interfaz adaptable a dispositivos móviles, tablets y computadoras de escritorio.

## 🛠️ Tecnologías Utilizadas

* **Framework:** [Vue.js](https://vuejs.org/) (Versión 3)
* **Herramienta de construcción:** [Vite](https://vitejs.dev/) (o Vue CLI, dependiendo de cómo lo iniciaste)
* **Lenguajes:** HTML5, CSS3, JavaScript (ES6+)
* **Estilos:** CSS puro / Tailwind CSS / Bootstrap *(modifica esto según lo que hayas usado)*
